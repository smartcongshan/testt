	1.文本编辑器、字处理器
		ASCII
	2.vim模式：
		(1)编辑模式(命令模式)
			所有的敲击键盘的命令会被理解为编辑整个文档(编辑功能)
		(2)输入模式
			大多数的敲键动作，会当做输入到文件中的内容，保存在文档当中。
		(3)末行模式
			输入很多除了编辑命令外的诸多文件管理类的命令
	3.模式转换：
		编辑模式―――>输入模式：
			i:在当前光标所在字符的前面，转为输入模式；下面转换为 --INSERT--
			a:在当前光标所在字符的后面，转为输入模式；下面转换为 --INSERT--
			o:在当前光标所在行的下方，新建一行，并将为输入模式		
			I:在当前光标所在行的行首，转换为输入模式
			A:在当前光标所在行的行尾，转换为输入模式
			O:在当前光标所在行的上方，新建一行，并将为输入模式
		输入模式――――>编辑模式：
			ESC键
		输入模式――――>末行模式：
			先由输入模式转换为编辑模式后再进入末行模式
		编辑模式――――>末行模式：
			：先输入冒号(:)
		末行模式――――>编辑模式
			ESC，ESC  敲击一次或者两次ESC即可
	4.Vim编辑器：
		全屏编辑器，模式化编辑器
		(1)打开文件
			# vim /path/to/somefile 后面有多个文件的时候，默认是打开一个文件
			vim +n FILE:打开文件，并定位置至第N行的行首
			vim + FILE:打开文件，并定位至最后一行的行首
			vim +/PATTERN:打开文件，定位至第一次被PATTERN匹配到的行的行首
			打开文件之后，默认是处在编辑模式之下的
		(2)关闭文件：一般是在末行模式或者编辑模式下来完成
			末行模式关闭文件：
				:w 保存(文件为只读文件是无法存，对root来说是可以强行保存的)
				:w! 以root身份强行保存
				:q 退出
				:wq  保存并退出  -->:x
				:q! 不保存并退出
			编辑模式下退出
				ZZ:键盘上连敲两下ZZ保存并退出
		(3)移动光标
			逐字符移动：(编辑命令下使用)
				h:向左移动光标   #h:移动#个字符
				l:向右移动光标
				j:向下移动光标
				k:向上移动光标
			以单词为单位移动：(编辑命令使用)
				w:移至下一个单词的词首			#w:移至下#个单词的词首
				e:移至当前或者下一个单词的词尾  #e
				b:跳至当前或者前一个单词的词首  #b
			行内跳转
				0:快速跳转至行首(绝对行首，最前)
				^:快速跳转至行首(相对行首，跳到第一个非空白字符)
				$:绝对行尾
			行间跳转
				#G:跳转至第#行
				G:跳转至最后的一行
				末行模式下跳转：:给出行号#，回车即可直接跳转
		(4)翻屏	
			Ctrl+f：向下翻一屏
			Ctrl+b：向上翻一屏
			Ctrl+d：向下翻半屏
			Ctrl+u：向上翻半屏
		(5)删除单个字符
			x:删除光标所在处的单个字符
			#x:删除光标所在处及向后的共#个字符
		(6)删除命令：d
			d命令+跳转命令，实现快速删除
			d$:从光标所在处一直删除到行尾
			d0:从光标所在处一直删除到行首
			dw:按单词逐个删除
			#d+跳转符:删除跳转符所指定的范围的#个
			#dw，#de，#db
			dd:删除当前光标所在行
			#dd:删除包括当前光标所在行之内的#行
			末行模式下删除：
				StartADD,EndADDd
				eg:删除第1行到第8行   :1,8d
				   从当前行，删除下面的五行(共6行)  :.,+5d
				   从当前行，删除到第100行    :.,100d
				   从当前行开始，删除到倒数第三行    :.,$-3d
				.:表示当前行
				$:表示最后一行  $-1:倒数第二行，$-10:倒数第十行
				+:向下的#行
		
2016.1.7
	1.vim所删除的内容不会立即被清空
		最近所删除的N个内容不会被立即清空，可以撤销之前所做的删除操作
		刚刚删除(最后一次删除)的内容还可以粘贴到别的位置。
		删除的内容会被保存在缓冲区中，缓冲区位于内存当中
	2.vim的编辑命令：
		(7)粘贴命令：p(编辑模式下输入)
			P：如果删除或复制为整行内容，则粘贴至光标所在行的上方；
			   如果删除或复制的内容为非整行，则粘贴至光标所在字符的前面
			p；如果删除或复制为整行内容，则粘贴至光标所在行的下方；
			   如果删除或复制的内容为非整行，则粘贴至光标所在字符的后面
		(8)复制命令：y (用法同d命令)
			yy:复制当前光标所在行
			#yy:复制包括当前光标所在行之内的#行
			y$:从光标所在处一直复制到行尾
			y0:从光标所在处一直复制到行首
			yw:按单词逐个复制
			#y+跳转符:复制跳转符所指定的范围的#个
			#yw，#ye，#yb
			末行模式下复制：
				StartADD,EndADDy
				eg:复制第1行到第8行   :1,8y
				   从当前行，复制下面的五行(共6行)  :.,+5y
				   从当前行，复制到第100行    :.,100y
				   从当前行开始，复制到倒数第三行    :.,$-3y
		(9)修改：先删除内容，再转换为输入命令 c
			c的用法同d命令
		(10)替换；将制定内容替换成某个字符，只能替换成单个字符 r
			rx:删除单个字符
			rh:将字符替换成h
			R:替换模式
				敲下大R直接进入替换模式，敲击键盘直接进行字符的替换，Ctrl+c 退出替换模式
		(11)撤销编辑操作 u(undo)
			u:撤销前一次的编辑操作
				可以连续使用u来恢复前面删除的内容
			#u:撤销#次的编辑操作
			撤销最近一次的撤销操作：ctrl+r  
		(12)重复前一次的编辑操作  .
			.命令
		(13)可视化模式 
			V:按举行块选取
				选取的是光标所在的整行
			v:按字符选取
				v+光标的移动方向：选取光标滑过的所有内容
		(14)查找
			/PATTERN:从文件首部向尾部查找
			?PATTERN:从文件尾部向首部查找
			N n  查找下一个
		(15)查找并替换(在末行模式下)
			使用s命令 等同于在nano模式下使用sed命令
			ADDR1,ADDR2s@PATTERN@string@gi
			地址1，地址2 s命令 Pattern String g全局 i忽略字符大小写
			eg：从光标所在的当前行，替换至倒数第二行，将he换位大写的HE 
				:.,$-1s@he@HE@g  (:.,$-1s/he/HE/g)
				1,$<--->%表示全文搜索
	3.vim编辑器会在内存当中存储50次的编辑操作的记录
		最多使用u命令撤销50次的编辑操作
